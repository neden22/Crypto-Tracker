# 🚀 Crypto Tracker

Premium crypto dashboard for monitoring cryptocurrency prices in real time.

This project was built as a portfolio project for frontend development practice and demonstrates working with APIs, dynamic UI rendering, filtering, sorting, and advanced UI interactions.

---

## ✨ Features

• Real-time cryptocurrency data via API  
• Live ticker with market updates  
• Search by coin name or symbol  
• Advanced sorting (price, market cap, change, rank)  
• Top gainers / losers filters  
• Favorites system with LocalStorage  
• Watchlist sidebar  
• Compare mode for two cryptocurrencies  
• Detailed modal view for each coin  
• Sparkline charts for price trends  
• Grid / List layout switch  
• Dark / Light theme toggle  
• Auto refresh every 60 seconds  
• Skeleton loaders while fetching data  
• Toast notifications  
• Premium neon UI with smooth animations

---

## 🧠 Tech Stack

Frontend:

- JavaScript (ES6+)
- SCSS
- Vite
- REST API

API:

- CoinGecko API

Browser APIs:

- LocalStorage
- Fetch API
